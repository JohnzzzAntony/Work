  - Finds all tasks where `status NOT IN ['done','closed']` AND `followUpFrequencyHours` is set, plus the first active admin.
  - For each task, computes `hoursSinceLastFollowUp = lastFollowUpAt ? (now - lastFollowUpAt)/1h : Infinity`.
  - If `hoursSinceLastFollowUp >= followUpFrequencyHours`: creates a `reminder` FollowUp (message `Follow-up reminder: <title> — please provide a status update`, sentToUserId = assigneeId), a `follow_up` notification to the assignee, updates the task (`lastFollowUpAt = now`, `followUpCount += 1`), and logs (`actionType: follow_up`, content `Follow-up #<count> sent to <assignee name>`).
  - Independently, if the task is also overdue (`dueDate < now`) AND `hoursSinceLastFollowUp >= escalationOverdueHours`: creates an `escalation` FollowUp (sentToUserId = admin), an `escalation` notification to the admin (`Escalation: <title> (assigned to <assignee>) is overdue`), and logs (`actionType: escalation`, content `Escalated to admin (overdue)`).
  - Returns the total count of FollowUp records created during this run.
